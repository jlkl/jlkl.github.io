和Laravel Debug RCE的区别？

日志中只出现一次完整payload，还有一次到15字符省略的payload

因为在utf-16le->utf-8的时候必须要求是偶数字节，不然会报错，所以在Laravel Debug RCE的时候会先发送一个AA的文件名让日志必定为偶数字节

在yii这里测试的时候本身就是偶字节的原因，就不需要提前发了，但是在调试Laravel的时候需要注意
